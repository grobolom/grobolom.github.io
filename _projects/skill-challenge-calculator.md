A Vue.js application that calculates success probabilities for skill challenges in Dungeons & Dragons 5th and 4th edition.

Skill challenges are a mechanic where players must accumulate a certain number of successes before failing a set number of times. This calculator helps DMs and players understand the actual odds of success given party composition and DC values.

**Features:**
- Calculate probability of success for any skill challenge configuration
- Support for both 5e and 4e rule variants
- Visual breakdown of outcomes

**Tech:** Vue.js
